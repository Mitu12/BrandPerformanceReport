# 📊 Brand Performance Analysis (SQL + Python + Reporting)

## 🔍 Project Overview
This project analyzes brand performance using SQL and Python, and presents insights through a business report.

---

## 🛠️ Tools & Technologies
- SQL (data extraction & transformation)
- Python (data processing)
- Excel / Reporting
- Data Visualization

---

- Sales Trend %
- Seen Rx
- Doctor Call Activity

- Analyzed sales trends across products
- Evaluated doctor engagement impact on performance
- Highlighted growth opportunities

This analysis helps:
